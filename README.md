## Selected Projects
### AI
- **MNIST Number Classifier (Keras, minimal)** — trains a compact MLP on MNIST and prints **only the predicted digit** (CLI). Clean data pipeline, model save/load, optional visualization.  
  *Stack:* Python, TensorFlow/Keras, NumPy. *Result:* ~98–99% test accuracy; fast CPU inference.

- **Linear Regression (C→F) — reproducible baseline** — single-neuron linear model with stable convergence to OLS; batch predictions for °C lists; quiet logs.  
  *Stack:* Python, TensorFlow/Keras, NumPy. *Result:* w≈1.80, b≈32.0; example outputs like `0°C→31.95°F`, `100°C→211.93°F`.
